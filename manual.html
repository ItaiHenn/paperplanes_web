6. עדכון תמונות ומדיה

להחליף תמונה באתר:

  1. כנסו ל- github.com/itaihenn/paperplanes_web
  2. נווטו לתיקיית assets/
  3. לחצו על "Add file" → "Upload files"
  4. גררו את התמונה החדשה עם אותו שם (למשל logo.jpg)
  5. לחצו "Commit changes" — הורסל יתעדכן אוטומטית

8. פתרון בעיות נפוצות

⚠️ הפרק לא מופיע באתר אחרי עדכון ב-Sheets
פתרון: טענו את הדף מחדש פעמיים (הפעם הראשונה מסנכרנת, השנייה מציגה).
⚠️ האתר מציג גרסה ישנה
פתרון: לחצו Ctrl+Shift+R (Windows) או Cmd+Shift+R (Mac) לרענון מלא.
🚨 האתר ריק / לא נטען כראוי
1. בדקו שה-Vercel deployment הצליח (vercel.com → הפרויקט → Deployments)
2. בדקו שה-Apps Script של Google Sheets עדיין מוגדר כ-Web App עם גישה ל-Anyone
3. פתחו F12 → Console ובדקו שגיאות JavaScript
💡 איך לוודא שהכל עובד:
כנסו לאתר ב-Incognito Mode — אם האתר עובד שם, הבעיה היא cache מקומי בדפדפן.
